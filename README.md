<H1>JavaFX and OOP Assignment</H1>

# Overview
Developing a user interface with menus, menu items, and a TableView using JavaFX and CSS, replicating the provided design.The interface contains a navigation menu, a profile image section, a central table displaying student information, and a form panel used to interact with the table data.

This project focuses on applying JavaFX layout design, styling with CSS, and implementing basic CRUD-style interactions within a graphical user interface.

Assignment for Software Engineering CSC 325

# Learning
1. Use of JavaFX Layout Management - VBox, BorderPane, StackPane etc.
2. Use of event handling
3. TableView and data models
4. CSS styling in JavaFX

# Conclusion
This project demonstrates the fundamentals of building structured JavaFX applications. It highlights how JavaFX layouts, TableView components, CSS styling, and event handling can work together to create an organized and interactive user interface.
